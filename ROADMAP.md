# TextReader 产品路线图

## 产品定位

TextReader 是一个跨平台（iOS / macOS）的离线文本阅读器，支持手动输入、文件导入和中文 TTS 朗读，逐步演进为“离线听书 / 学习朗读助手”。

---

## 阶段目标总览

- **Phase 0：稳定可用**
  - 完成输入、导入、朗读、停止的可靠闭环
  - 保证跨平台基础体验一致
- **Phase 1：MVP 可发布**
  - 增加暂停/继续、设置页、状态记忆
  - 提升“愿意长期使用”的体验
- **Phase 2：1.0 正式版**
  - 做听读同步、高亮、章节识别、搜索
  - 从工具升级为阅读产品
- **Phase 3：1.x 增强版**
  - 扩展更多格式、后台播放、音频导出、资料库
- **Phase 4：2.0 探索版**
  - OCR、网页朗读、AI 辅助阅读、跨设备同步

---

## Phase 0：稳定可用（当前重点）

### 目标
让用户第一次打开就能无障碍完成“输入 / 导入 / 朗读 / 停止”。

### 核心任务
- [x] 手动输入文本
- [x] 导入 `.txt` 文本文件
- [x] 开始 / 停止朗读
- [x] 跨平台基础兼容（iOS / macOS）
- [x] Preview 独立运行
- [ ] 更稳定的长文本分句与停止响应
- [ ] 更清晰的错误提示（不要把错误信息混入正文）
- [ ] TTS 引擎故障时自动回退到系统语音

### 验收标准
- 输入一段文本后可以顺利朗读
- 导入文件后可以顺利朗读
- 停止操作及时生效
- 界面不会因 TTS 初始化失败而不可用

## Phase 1：MVP 可发布版

### 目标
让产品从“Demo”升级为“可持续使用的工具”。

### 核心功能
#### 1. 阅读控制增强
- [x] 暂停 / 继续
- [ ] 上一句 / 下一句
- [ ] 从当前位置继续播放
- [ ] 记住上次朗读位置

#### 2. 设置页
- [x] 切换语音引擎（系统 / Piper）
- [x] 调整语速
- [x] 调整音量
- [x] 调整音调
- [ ] 选择具体音色（当模型支持时）

#### 3. 内容持久化
- [ ] 自动保存当前文本
- [ ] 最近打开文件
- [ ] 恢复上次设置
- [ ] 恢复上次阅读内容

#### 4. 阅读体验优化
- [ ] 字体大小调整
- [ ] 行距调整
- [ ] 深色模式优化
- [ ] 更适合长文阅读的排版

### 验收标准
- 用户可以方便控制朗读节奏
- 设置修改后立即生效或对下一次朗读生效
- 下次打开应用时不会完全从零开始

## Phase 2：1.0 正式版

### 目标
建立“听读同步”的核心产品体验。

### 核心功能
#### 1. 当前句高亮
- [ ] 正在朗读的句子高亮
- [ ] 停止后保留高亮位置
- [ ] 点击某句从该句开始读

#### 2. 自动定位
- [ ] 朗读时滚动到当前句
- [ ] 可关闭自动定位
- [ ] 用户手动滚动时不强制打断

#### 3. 章节识别与目录
- [ ] 识别“第一章 / 第二章 / 第一卷”等标题
- [ ] 自动生成目录
- [ ] 点击目录跳转
- [ ] 从当前章节开始朗读

#### 4. 文本搜索
- [ ] 关键字搜索
- [ ] 搜索结果定位
- [ ] 从搜索结果位置开始朗读

### 验收标准
- 朗读不再只是“播音”，而是“可跟随的阅读过程”
- 长文本结构化阅读成为产品优势

---

## Phase 3：1.x 增强版

### 目标
提高留存，覆盖更真实的阅读场景。

### 核心功能
#### 1. 更多格式支持
- [ ] Markdown (`.md`)
- [ ] RTF (`.rtf`)
- [ ] EPUB (`.epub`)
- [ ] PDF (`.pdf`)

#### 2. 阅读历史 / 资料库
- [ ] 最近阅读
- [ ] 收藏文本
- [ ] 文件分类管理
- [ ] 历史记录和继续阅读入口

#### 3. 导出音频
- [ ] 导出为 `.wav`
- [ ] 导出为 `.m4a`
- [ ] 长文本分段导出

#### 4. 后台播放
- [ ] iOS 锁屏控制
- [ ] 控制中心暂停 / 继续
- [ ] 耳机快捷控制

#### 5. 中文朗读质量提升
- [ ] 数字、日期、英文混读更自然
- [ ] 标点停顿优化
- [ ] 对话和章节朗读更拟人化
